Tables are one of the very useful and important tools in HTML.
They are created using the <TABLE> tag.
There are multiple tags that can be used inside the table tag. These are as follows:
  1. <tr>
    - This tag is used to create Table Rows in HTML.
  2. <td>
    - This tag is used to fill data in HTML.
  3. <th>
    - This tag is used to create Table Headings in HTML.
  4. <Caption>
    - This tag is used to give caption to the HTML Table.
  5. <THEAD>
    - This tag is used to wrap up Table header.
  6. <TBODY>
    - This tag is used to wrap up Table body.
